Angular | Todo App

Todo App est une application de gestion de tache realisee avec le Framework Angular.

Pourquoi avoir cree une Todo App ?

Avec l'idee de comprendre le fonctionnement basique du FrameWork Angular, une petite Application de gestion de tache est la mieux approprie. Cette application m'a permit de comprendre les concerpts et l'environement d'Angular.

Environnment et concerpt
  1. Le passage d'un env de developpement a un env de test pour terminer avec un env de production
  2. Une connaissance sur l'architecture d'une App Angular.
  3. La comprehension des Composants Angular
  4. La comprehension des Directives Angular
  5. La connaissance sur la communication inter-composants Angular
    1. Communication Parent-Enfant
    2. Communication Enfant-Parent
